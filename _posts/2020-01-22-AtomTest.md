1. 본인 github.io repo를 open in desktop으로 clone한다.
![1. repo클론](/assets/1.%20repo클론.JPG)

- 자신의 github.io 폴더의 위치를 기억하자. clone후에 show in exploer 버튼을 누르면 해당 폴더를 열어준다.
![1. show in exploer](/assets/1.%20show%20in%20exploer.JPG)

2. Atom을 다운 받는다. 주소 : atom.io
![1. atom 다운받기](/assets/1.%20atom%20다운받기.JPG)

3. 패키지를 다운 받는다.
3-1. insatll package를 누른다.
![install package](/assets/install%20package.JPG)

3-2. 다운받을 패키지 목록
![다운받을 패키지](/assets/다운받을%20패키지.JPG)

4. file - add project folder를 누른다
![add project](/assets/add%20project.jpg)

5. 1에서 clone했던 github.io 폴더를 선택한다.

6. _posts 폴더에 들어가서 글을 쓴다.
- 이제부터 우측에 preview가 나오며 이미지를 드래그를 통해 삽입할 수 있습니다.
![preview](/assets/preview.png)
![이미지 드래그 삽입](/assets/이미지%20드래그%20삽입.png)

* 단, 파일 형식은 꼭 .md (markdown) 형식이어야 합니다.

7. 해당 내용은 github desktop으로 commit한다.
![commit](/assets/commit.JPG)
